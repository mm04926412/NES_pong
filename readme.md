# NES Pong (2019)

Pong programmed for the NES using NESASM (6502)

## Build and Run
(Requires NESASM3 compiler and an NES emulator)

  - run `nesasm3 pong.asm` in powershell
  - Load the outputted `.NES` file into an emulator
